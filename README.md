# 📦 Amazon Sales Analysis Dashboard | Tableau

## 📌 Overview

This project presents an **interactive Tableau dashboard** analyzing Amazon sales data to uncover insights across revenue, orders, cost, and sales channels.

The dashboard is designed for **executive-level decision-making**, helping stakeholders quickly understand performance trends and optimize business strategies.

Such dashboards are commonly used in analytics to track **sales performance, profitability, and operational efficiency** ([LinkedIn][1])

## 🚀 Key Features

* 📊 **Revenue by Channel Analysis** (Online vs Offline)
* 📈 **Orders Trend by Year**
* 💰 **Cost Analysis Over Time**
* ⚡ **Order Priority Distribution (High, Medium, Low, Critical)**
* 🎛️ **Interactive Filters** (Year, Item Category)
* 📉 Clean, executive-friendly dashboard layout

---

## 💡 Key Insights

* Online sales contribute **~60% of total revenue**, outperforming offline channels
* Orders peak around **2012**, followed by a gradual decline
* Cost trends closely follow order volume → indicating operational dependency
* High-priority orders dominate → showing urgent demand patterns
* Opportunity exists for **cost optimization and demand forecasting**
